# avengers-social-network

## Possible flows

### User
1. create user 
2. update user 
3. get users
4. delete user 
5. follow a user
6. poke a user
7. update/delete bio 
8. change/delete profile picture
9. upload + set profile picture

### Posts
1. create a post 
2. get posts (by user)
3. like a post 
4. bookmark a post 
5. repost a post 
6. get trending posts 
(top 50 with most interactions in the last 24 hours)


### Events 
1. create an event 
2. update an event 
3. get events 
4. cancel an event 
5. enroll in an event (as a user)
